## [ DEVELOPMENT WILL BEGIN IN AUGUST 2017 ]
- The goal is to translate the same SQL code from one database server language to another database server language:
 - T-SQL -> Oracle
 - Oracle -> T-SQL
 - T-SQL -> PostgreSQL
 - PostgreSQL -> T-SQL
 - T-SQL -> MySQL
 - MySQL-> T-SQL
